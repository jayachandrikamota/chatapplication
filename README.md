# 💬 Real-Time Chat Application (C++)

A real-time multi-client chat application developed in **C++** using **Winsock**, **TCP socket programming**, and **multithreading**. The application enables multiple users to communicate simultaneously through a client-server architecture.

## 🚀 Features

- Real-time messaging
- Multi-client communication
- Client-server architecture
- TCP socket programming
- Multi-threaded server
- Private messaging
- Chat room support
- Chat history
- Search messages
- Thread synchronization using mutexes

## 🛠️ Technologies Used

- C++
- Winsock API
- TCP/IP
- Socket Programming
- Multithreading
- Mutex Synchronization

## ⚙️ How It Works

1. Start the server.
2. Launch one or more client applications.
3. Clients connect to the server using TCP sockets.
4. Users can send and receive messages in real time.
5. The server manages multiple clients simultaneously using threads.
